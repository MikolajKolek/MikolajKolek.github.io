Building the project

You need to have the development enviroment already set up for this stage. To build the project simply click File>Build Settings with the project open in Unity Editor.
Then select the "PC, Max & Linux Standalone" platform and set the "Target Platform" field to windows.
Then press the "Build" button and select a folder in which you want to build the game (we recommend using the Builds folder as it contains the icon and licence for the installer). Lastly, paste the "lang" and "levels" folders to the Builds folder. The end result should look similar to this:
