Detailed Description

Acts as an adapter between the Tilemap and Grid and the game. It has the ability to load and properly scale levels, move the player, restart the game and call delegates when certain events happen. It has improvements over the base TilemapGameAdapter that add a capability to save and load game data to it, and also to be used for new level playtesting.
